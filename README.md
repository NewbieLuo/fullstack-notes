# fullstack-notes

AI-native 随堂作业：monorepo + 前后端分离 + Supabase + GitHub CI/CD 全栈笔记应用。

## Stack
- pnpm workspaces + Turborepo
- Next.js 15 (static export → GitHub Pages)
- Hono (→ Vercel Serverless)
- Supabase (Postgres)
- TypeScript + Biome + Vitest
